Help Desk Angular with Spring boot

Este projeto foi desenvolvido como parte da minha formação no curso Desenvolvedor Full Stack. Trata-se de uma aplicação completa para gerenciamento de chamados, construída com uma arquitetura moderna utilizando Angular 17 no frontend e Spring Boot 3.2.5 no backend.

Destaques do Projeto:

  • Arquitetura: Aplicação SPA (Single Page Application).
  • Segurança: Implementação de Autenticação e Autorização via tokens JWT.
  • Cloud: Frontend hospedado na Vercel e backend no Heroku.

O código-fonte está disponível no link GitHub no menu lateral. Sinta-se à vontade para explorar o sistema e enviar suas sugestões!
